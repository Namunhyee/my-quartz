# nidogen

![[Cell Biology/출처/25년 강의노트/md_converted/Ch19-2_Cell_Junctions_and_the_Extracellular_Matrix/_page_39_Figure_1.jpeg]]
*Basal lamina의 분자 구조 모델. Nidogen (entactin이라고도 함)이 laminin과 type IV collagen을 cross-link하여 basal lamina의 구조적 integrity를 제공한다.*

**Nidogen** (또는 **entactin**)은 [[Basal Lamina_MOC|basal lamina]]의 glycoprotein으로, [[Laminin|laminin]]과 type IV collagen을 연결하는 핵심 cross-linker이다.

## 기능

- **Laminin γ chain**에 결합
- **Type IV collagen**에 결합
- 두 주요 basal lamina 구성요소를 **cross-link**하여 연결
- [[Proteoglycan|Perlecan]]과도 상호작용

Laminin, type IV collagen, nidogen, perlecan이 함께 상호작용하여 **고도로 cross-linked된 단백질 및 proteoglycan network**를 형성하며, 이것이 basal lamina의 구조와 기능을 결정한다.

## 유전적 연구 결과

흥미롭게도, nidogen-1 유전자를 knockout한 쥐나 nidogen-2 유전자를 knockout한 쥐는 모두 건강하고 정상적인 표현형을 보인다. 그러나 laminin γ chain에서 nidogen 결합 부위만을 제거한 돌연변이 쥐는 출생 시 사망하며 폐와 신장 형성에 심각한 결함을 보인다. 이는 nidogen이 중복 기능(redundancy)을 가지거나, laminin의 nidogen 결합 domain이 다른 기능도 가지고 있음을 시사한다.
