# Step 3 - Phosphofructokinase

## 반응

**Fructose-6-phosphate + ATP → Fructose-1,6-bisphosphate + ADP**

```
Fructose-6-phosphate + ATP  ──(PFK-1)──→  Fructose-1,6-bisphosphate + ADP
```

## 핵심 정보

| 항목 | 내용 |
|------|------|
| **Reactant** | Fructose-6-phosphate, ATP |
| **Product** | Fructose-1,6-bisphosphate (F1,6-bisP), ADP |
| **Enzyme** | Phosphofructokinase-1 (PFK-1) |
| **Phase** | Energy investment |
| **ATP** | −1 (소비) |

## 특징

- **Glycolysis의 가장 중요한 조절 효소** (rate-limiting step)
- **비가역 반응** → 주요 조절 지점
- **억제 인자**: ATP (에너지 과다), Citrate (TCA cycle 중간체), NADH
- **활성화 인자**: AMP, ADP (에너지 부족), Fructose-2,6-bisphosphate (인슐린 신호)
