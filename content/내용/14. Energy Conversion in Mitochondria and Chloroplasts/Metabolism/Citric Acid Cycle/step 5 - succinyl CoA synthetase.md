# Step 5 - Succinyl CoA Synthetase

## 반응

**Succinyl CoA + GDP + Pᵢ → Succinate + GTP + CoA**

```
Succinyl CoA + GDP + Pᵢ  ──(Succinyl CoA synthetase)──→  Succinate + GTP + CoA
```

## 핵심 정보

| 항목 | 내용 |
|------|------|
| **Reactant** | Succinyl CoA, GDP, Pᵢ |
| **Product** | Succinate, GTP, CoA |
| **Enzyme** | Succinyl CoA synthetase |
| **산물 에너지** | GTP ≈ ATP (substrate-level phosphorylation) |

## 특징

- **Substrate-level phosphorylation**: TCA cycle에서 유일하게 직접 고에너지 인산 결합 생성
- GTP 생성 (일부 조직에서는 ATP 직접 생성)
- GTP는 ATP와 동등한 에너지를 가짐 (GTP + ADP ↔ GDP + ATP, nucleoside diphosphate kinase)
- CoA 방출 → 재사용
- Thioester 결합의 에너지로 phosphorylation 구동
