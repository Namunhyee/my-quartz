# Step 2 - Aconitase

## 반응

**Citrate → Isocitrate** (via cis-aconitate)

```
Citrate  ──(Aconitase)──→  cis-Aconitate  ──→  Isocitrate
         (탈수)                              (수화)
```

## 핵심 정보

| 항목 | 내용 |
|------|------|
| **Reactant** | Citrate |
| **Product** | Isocitrate |
| **Enzyme** | Aconitase (iron-sulfur protein) |
| **Cofactor** | [4Fe-4S] iron-sulfur cluster |
| **산물 에너지** | 없음 |

## 특징

- **이성질화**: Citrate (tertiary alcohol)를 반응성 높은 Isocitrate (secondary alcohol)로 전환
- 두 단계: 탈수 → 수화 (net: OH기 위치 이동)
- Aconitase는 **iron-sulfur cluster** 포함 → 효소 활성에 중요
- 가역 반응이나 isocitrate 쪽이 선호

> 교과서: iron-sulfur clusters는 ETC에서도 중요한 역할 (Complex I, II)
