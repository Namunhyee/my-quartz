# 3. Hydroxyacyl CoA Dehydrogenase

## 반응

**L-3-Hydroxyacyl CoA + NAD⁺ → 3-Ketoacyl CoA + NADH + H⁺**

```
L-3-Hydroxyacyl CoA + NAD⁺  ──(Hydroxyacyl CoA dehydrogenase)──→  3-Ketoacyl CoA + NADH
```

## 핵심 정보

| 항목 | 내용 |
|------|------|
| **Reactant** | L-3-Hydroxyacyl CoA, NAD⁺ |
| **Product** | 3-Ketoacyl CoA, NADH |
| **Enzyme** | L-3-Hydroxyacyl CoA dehydrogenase |
| **Cofactor** | NAD⁺ |
| **산물 에너지** | NADH (→ ETC Complex I, ~2.5 ATP) |

## 특징

- **두 번째 산화 반응**: 3번 탄소의 hydroxyl기를 keto기로 산화
- NADH 생성 → 미토콘드리아 ETC로 전자 전달
- 3-Ketoacyl CoA는 이제 다음 thiolysis 반응에 취약한 상태
