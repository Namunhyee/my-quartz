# 해설

광합성 명반응 핵심 개념:

**반응 위치**: 틸라코이드 막(thylakoid membrane)

**광계 II (PSII) → 광계 I (PSI) 전자 흐름 (Z-scheme):**
1. **PSII**: 빛 에너지 흡수 → 물 분해(H₂O → 2e⁻ + 2H⁺ + ½O₂) → 전자 방출
2. **플라스토퀴논 (PQ)**: PSII → Cytochrome b₆f complex로 전자 전달
3. **Cytochrome b₆f**: H⁺를 틸라코이드 내강(lumen)으로 펌핑 (미토콘드리아 Complex III 유사)
4. **플라스토시아닌(PC)**: b₆f → PSI로 전자 이동
5. **PSI**: 빛 에너지로 전자를 더 높은 에너지 준위로 높임 → Ferredoxin → **NADP⁺ → NADPH** 합성

**ATP 합성:**
- H⁺가 틸라코이드 lumen에 축적 → 막전위 형성
- **CF₀CF₁ ATP synthase**를 통해 stroma 방향으로 H⁺ 이동 → ATP 합성

**미토콘드리아 ETC와 비교:**
| | 미토콘드리아 | 엽록체 |
|-|-|-|
| H⁺ 고농도 구역 | IMS | 틸라코이드 lumen |
| ATP 합성 방향 | IMS → matrix | lumen → stroma |
| 최종 전자 수용체 | O₂ (→ H₂O) | NADP⁺ (→ NADPH) |
