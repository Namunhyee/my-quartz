# 31번

## 발문

**Talin에 직접적으로 binding할 수 있는 단백질을 3가지 쓰시오.** (E, 3점)

## 관련 내용

- [[Cell Biology/내용/19. Cell Junctions and the Extracellular Matrix/Cell Junctions/Cell-Matrix Junctions/integrin/Related proteins/talin]]

# 해설

**정답: Integrin (β subunit), Vinculin, Actin (F-actin)**

---

**Talin의 구조와 결합 파트너:**

Talin은 대형 scaffold 단백질 (~270 kDa):
- **Head domain (FERM domain)**: integrin 결합 부위
- **Rod domain**: vinculin 결합 부위(다수의 VBS: vinculin binding site), actin 결합 부위

**직접 결합 단백질 3가지:**

1. **Integrin (β 서브유닛 cytoplasmic tail)**
   - Talin head의 F3 subdomain이 integrin β 꼬리와 결합
   - → **inside-out signaling** 유발 → integrin 활성화(high-affinity state)

2. **Vinculin (빈쿨린)**
   - Talin rod domain의 VBS(vinculin binding site)에 결합
   - Vinculin은 actin과도 결합 → focal adhesion 강화·안정화

3. **F-actin (actin filament)**
   - Talin rod domain에 actin-binding site 존재
   - Talin이 integrin과 actin을 직접 연결 → mechanotransduction 전달

**추가 결합 파트너 (직접):**
- **PIP₂ (phosphatidylinositol 4,5-bisphosphate)**: talin의 autoinhibition 해제
- **RIAM (Rap1-GTP-interacting adaptor molecule)**: talin 활성화 촉진

**Focal adhesion에서의 역할:**
```
ECM ← Integrin ← Talin ← Vinculin ← Actin → 세포골격
```
Talin이 integrin과 actin 세포골격 사이의 **핵심 기계적 연결(linker)** 역할
