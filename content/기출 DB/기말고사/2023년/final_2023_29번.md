# 해설

**정답: Serine (세린, Ser)**

---

**GAG chain 연결 구조:**

```
Core protein — Ser — (O-xylose) — GalNAc/GlcNAc 반복 ... → GAG chain
```

**상세 구조:**
1. Core protein의 **Ser-Gly-X-Gly 모티프**의 세린(Ser)에 O-연결(O-glycosidic bond)로 연결
2. **Tetrasaccharide linker**: Xylose(Xyl) — Galactose(Gal) — Galactose(Gal) — Glucuronic acid(GlcUA)
3. 이 linker에 이어서 반복 이당류 단위(GAG)가 추가

**GAG 종류별 core protein 연결 아미노산:**
- **Chondroitin sulfate, Dermatan sulfate, Heparan sulfate, Heparin**: 모두 **Serine(Ser)** — O-glycosidic bond (via xylose)
- **Keratan sulfate**: Serine 또는 **Threonine(Thr)** — O-glycosidic bond, 또는 Asparagine(Asn) — N-glycosidic bond

> 문제 그림이 일반적인 proteoglycan 구조를 보여준다면 정답은 **Serine**.

**Proteoglycan 예시:**
- Aggrecan (연골): chondroitin sulfate + keratan sulfate
- Syndecan (세포 표면): heparan sulfate
- Perlecan (기저막): heparan sulfate
