# 해설

**정답: ④ B, D, A, C**

Glycolysis 전체 순서:

1. Glucose → Glucose-6-phosphate (Hexokinase, ATP 소모)
2. → Fructose-6-phosphate
3. → **Fructose 1,6-bisphosphate (B)** (Phosphofructokinase, ATP 소모)
4. → **Dihydroxyacetone phosphate (D)** + Glyceraldehyde-3-phosphate (Aldolase)
   - DHAP ↔ G3P (Triose phosphate isomerase)
5. G3P → **1,3-bisphosphoglycerate (A)** (Glyceraldehyde-3-phosphate dehydrogenase, NADH 생성)
6. → 3-phosphoglycerate (ATP 생성)
7. → 2-phosphoglycerate
8. → **Phosphoenolpyruvate (C)** (Enolase)
9. → Pyruvate (Pyruvate kinase, ATP 생성)

**순서: B → D → A → C ④**
