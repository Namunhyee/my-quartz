# 해설

**정답: ⑤**

세포막 특성 분석:

① 막단백질은 세포막에 박혀 이동하지 않는다. → 틀림 ✗  
막단백질은 막에 박혀 있기는 하나, 많은 경우 막 평면 내에서 측방 확산이 가능함. 즉 완전히 고정된 구조는 아님.

② 막단백질은 특정 도메인을 건너서 이동할 수 있다. → 틀림 ✗  
상피세포처럼 막 도메인(apical / basolateral)이 나뉜 경우, tight junction이 경계(fence) 역할을 하여 막단백질의 이동을 제한함.

③ 막단백질은 안과 바깥쪽의 지질 이중층을 옮겨 다닐 수 있다. → 틀림 ✗  
막단백질은 한 번 삽입되면 막의 방향성(topology) 이 유지됨. 지질처럼 leaflet 사이를 오가는 것도 아니고, 막단백질이 flip-flop 하는 것도 사실상 일어나지 않음.

④ Tight junction으로 막단백질과 인지질의 이동이 제한될 수 있다. → 맞음 ✓  
Tight junction은 단순한 세포 간 접착 구조가 아니라, 막 성분의 측방 이동을 막는 울타리 역할도 함. 따라서 막단백질과 일부 지질의 도메인 간 이동이 제한될 수 있음.

⑤ 도메인 사이의 단백질은 확산이 방지되나 지질은 계속 확산이 가능하다. → 틀림 ✗  
Tight junction은 단백질만이 아니라 지질의 확산도 제한할 수 있음. 따라서 “지질은 계속 자유롭게 확산 가능”이라는 서술은 부정확함.
