# 28번

## 발문

인지질 이동(phospholipid translocation)을 담당하는 효소들을 빈칸에 채우시오.

| 효소 | 방향 | ATP 의존성 | 특이성 |
|------|------|-----------|--------|
| __(A)__ | outer → inner | ATP 의존 | PS, PE 특이적 |
| __(B)__ | inner → outer | ATP 의존 | PC, SM 특이적 |
| __(C)__ | 양방향 | ATP 비의존 | 비특이적 |

# 해설

**빈칸 정답:**

- **(A): Flippase** (P4-ATPase 계열)
  - 방향: outer leaflet → inner leaflet
  - ATP 의존적 (능동 수송)
  - PS, PE를 내엽에 유지 → 세포막 비대칭성 확립·유지
  
- **(B): Floppase** (ABC transporter 계열, 예: ABCA1)
  - 방향: inner leaflet → outer leaflet
  - ATP 의존적
  - PC, SM 등을 외엽으로 이동

- **(C): Scramblase**
  - 방향: 양방향 (무작위)
  - ATP 비의존적 (수동)
  - 비특이적 — 모든 인지질 이동
  - 활성화: ER에서 기저 활성(막 비대칭성 없음), apoptosis 시 Ca²⁺에 의해 활성화 → PS 외엽 노출

**세 효소의 역할 정리:**
- **정상 세포**: Flippase가 PS·PE를 내엽으로 유지 → 비대칭성 유지
- **Apoptosis**: Scramblase 활성화 → PS 외엽 노출 → "eat me" 신호
- **ER**: Scramblase 기저 활성 → 양 리플렛 균등 분포 (비대칭 없음)
