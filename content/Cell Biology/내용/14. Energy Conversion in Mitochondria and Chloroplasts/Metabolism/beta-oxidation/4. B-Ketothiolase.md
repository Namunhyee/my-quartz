# 4. β-Ketothiolase

## 반응

**3-Ketoacyl CoA + CoA → Acetyl CoA + (n-2 Acyl CoA)**

```
3-Ketoacyl CoA + CoA-SH  ──(β-Ketothiolase)──→  Acetyl CoA + Acyl CoA (탄소 2개 짧아진)
```

## 핵심 정보

| 항목 | 내용 |
|------|------|
| **Reactant** | 3-Ketoacyl CoA, CoA-SH |
| **Product** | Acetyl CoA, (단축된) Acyl CoA |
| **Enzyme** | β-Ketothiolase (Thiolase) |
| **산물 에너지** | Acetyl CoA → Citric Acid Cycle |

## 특징

- **Thiolysis**: CoA가 3-Ketoacyl CoA를 공격하여 Acetyl CoA를 절단
- 매 사이클마다 탄소 **2개** 제거 (Acetyl CoA 형태)
- 새로운 (단축된) Acyl CoA는 다시 Step 1로 → 사이클 반복
- **마지막 사이클**: 탄소 4개의 Acyl CoA → Acetyl CoA **2개** 생성

## 사이클 반복 종료 조건

- **짝수 탄소 지방산**: Acetyl CoA만 생성 (깔끔하게 종료)
- **홀수 탄소 지방산**: 마지막에 Propionyl CoA (C3) 생성 → 별도 대사 경로
