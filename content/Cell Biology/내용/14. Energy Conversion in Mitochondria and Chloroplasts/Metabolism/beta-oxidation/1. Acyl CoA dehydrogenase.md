# 1. Acyl CoA Dehydrogenase

## 반응

**Acyl CoA + FAD → trans-2-Enoyl CoA + FADH₂**

```
Acyl CoA + FAD  ──(Acyl CoA dehydrogenase)──→  trans-Enoyl CoA + FADH₂
```

## 핵심 정보

| 항목 | 내용 |
|------|------|
| **Reactant** | Acyl CoA (fatty acyl CoA), FAD |
| **Product** | trans-2-Enoyl CoA, FADH₂ |
| **Enzyme** | Acyl CoA dehydrogenase |
| **Cofactor** | FAD (flavin adenine dinucleotide) |
| **산물 에너지** | FADH₂ (→ ETC, ~1.5 ATP) |

## 특징

- Beta-oxidation의 **첫 번째 단계** (산화 반응)
- Acyl CoA의 α-β 탄소 사이에 **이중결합** 형성 (trans configuration)
- FAD는 단백질에 단단히 결합되어 있음 → 전자를 직접 ubiquinone으로 전달 (ETC Complex II와 유사)
- Acyl CoA의 탄소 길이에 따라 다른 isoform 존재 (short/medium/long chain)

## FADH₂의 의미

Beta-oxidation에서 FAD를 electron acceptor로 사용하는 이유: 이 산화 반응의 ΔG가 작아서 NAD⁺를 환원시키기에 충분하지 않다.
