# 2. Enoyl CoA Hydratase

## 반응

**trans-2-Enoyl CoA + H₂O → L-3-Hydroxyacyl CoA**

```
trans-Enoyl CoA + H₂O  ──(Enoyl CoA hydratase)──→  L-3-Hydroxyacyl CoA
```

## 핵심 정보

| 항목 | 내용 |
|------|------|
| **Reactant** | trans-2-Enoyl CoA, H₂O |
| **Product** | L-3-Hydroxyacyl CoA |
| **Enzyme** | Enoyl CoA hydratase (Crotonase) |
| **산물 에너지** | 없음 |

## 특징

- **수화 반응**: 이중결합에 물 분자 첨가
- L-입체이성질체 형성 (L-3-hydroxy)
- 이 반응 자체는 에너지 보존 없이 다음 산화 반응을 위한 **substrate 준비**
- 가역 반응 (열역학적으로 L-isomer가 선호)
