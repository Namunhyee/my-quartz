# Step 4 - α-Ketoglutarate Dehydrogenase

## 반응

**α-Ketoglutarate + NAD⁺ + CoA → Succinyl CoA + CO₂ + NADH**

```
α-Ketoglutarate + NAD⁺ + CoA  ──(α-KG dehydrogenase)──→  Succinyl CoA + CO₂ + NADH
```

## 핵심 정보

| 항목 | 내용 |
|------|------|
| **Reactant** | α-Ketoglutarate (C5), NAD⁺, CoA |
| **Product** | Succinyl CoA (C4), CO₂, NADH |
| **Enzyme** | α-Ketoglutarate dehydrogenase complex |
| **산물 에너지** | NADH (~2.5 ATP) |

## 특징

- **두 번째 산화적 탈탄산 반응**: CO₂ 방출 + NADH 생성
- 두 번째 탄소 제거 (C5 → C4), 이 시점에서 Acetyl CoA의 탄소 2개 모두 CO₂로 방출됨
- Pyruvate dehydrogenase complex와 **유사한 메커니즘** (multi-enzyme complex, thiamine pyrophosphate 사용)
- 비가역 반응
- **억제**: NADH, Succinyl CoA (product inhibition)
