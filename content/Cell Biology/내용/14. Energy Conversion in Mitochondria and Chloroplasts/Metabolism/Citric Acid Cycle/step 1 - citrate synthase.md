# Step 1 - Citrate Synthase

## 반응

**Acetyl CoA + Oxaloacetate + H₂O → Citrate + CoA**

```
Acetyl CoA + OAA  ──(Citrate synthase)──→  Citrate + CoA-SH
```

## 핵심 정보

| 항목 | 내용 |
|------|------|
| **Reactant** | Acetyl CoA (C2), Oxaloacetate (C4) |
| **Product** | Citrate (C6), CoA-SH |
| **Enzyme** | Citrate synthase |
| **산물 에너지** | 없음 (사이클 시작 반응) |

## 특징

- **사이클 입구**: Acetyl CoA의 C2와 OAA의 C4가 결합 → C6 citrate 형성
- 이 응축 반응은 thermodynamically favorable (비가역)
- **조절 효소**: NADH, ATP, Succinyl CoA에 의해 억제 (에너지 과다 시)
- CoA 방출 → 다음 Acetyl CoA 생성에 재사용
