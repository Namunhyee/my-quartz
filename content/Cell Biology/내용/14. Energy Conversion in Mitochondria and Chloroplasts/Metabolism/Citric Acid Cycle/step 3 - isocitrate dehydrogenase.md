# Step 3 - Isocitrate Dehydrogenase

## 반응

**Isocitrate + NAD⁺ → α-Ketoglutarate + CO₂ + NADH**

```
Isocitrate + NAD⁺  ──(Isocitrate dehydrogenase)──→  α-Ketoglutarate + CO₂ + NADH
```

## 핵심 정보

| 항목 | 내용 |
|------|------|
| **Reactant** | Isocitrate (C6), NAD⁺ |
| **Product** | α-Ketoglutarate (C5), CO₂, NADH |
| **Enzyme** | Isocitrate dehydrogenase |
| **산물 에너지** | NADH (~2.5 ATP) |

## 특징

- **첫 번째 산화적 탈탄산 반응**: CO₂ 방출 + NADH 생성
- 첫 번째 탄소 제거 단계 (C6 → C5)
- **조절 효소**: ADP에 의해 활성화, ATP·NADH에 의해 억제
- 비가역 반응
