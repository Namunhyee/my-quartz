# Step 7 - Fumarase

## 반응

**Fumarate + H₂O → L-Malate**

```
Fumarate + H₂O  ──(Fumarase)──→  L-Malate
```

## 핵심 정보

| 항목 | 내용 |
|------|------|
| **Reactant** | Fumarate (C4), H₂O |
| **Product** | L-Malate (C4) |
| **Enzyme** | Fumarase (Fumarate hydratase) |
| **산물 에너지** | 없음 |

## 특징

- **수화 반응**: 이중결합에 H₂O 첨가 → L-Malate 형성
- 입체특이적: trans(Fumarate)에서 L-Malate만 생성
- 가역 반응 (L-malate 쪽 선호)
- TCA cycle의 다음 마지막 산화를 위한 기질 준비
