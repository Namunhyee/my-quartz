# Step 1 - Hexokinase

## 반응

**Glucose + ATP → Glucose-6-phosphate + ADP**

```
Glucose  ──(Hexokinase)──→  Glucose-6-phosphate
   + ATP                          + ADP
```

## 핵심 정보

| 항목 | 내용 |
|------|------|
| **Reactant** | Glucose, ATP |
| **Product** | Glucose-6-phosphate (G6P), ADP |
| **Enzyme** | Hexokinase (Glucokinase in liver) |
| **Phase** | Energy investment |
| **ATP** | −1 (소비) |

## 특징

- **ATP 소비**: 인산기를 Glucose에 붙여 세포 밖으로 빠져나가지 못하게 함
- **Product inhibition**: G6P 농도 증가 → Hexokinase 억제 (음성 되먹임)
- 간에서는 **Glucokinase** (low affinity, no inhibition by G6P) — 혈당 높을 때 작동
- 비가역 반응 → 조절 지점
