# Step 6 - Glyceraldehyde 3-Phosphate Dehydrogenase

## 반응

**G3P + NAD⁺ + Pᵢ → 1,3-bisphosphoglycerate + NADH + H⁺**

```
G3P + NAD⁺ + Pᵢ  ──(G3PDH)──→  1,3-bisphosphoglycerate + NADH
```

## 핵심 정보

| 항목 | 내용 |
|------|------|
| **Reactant** | Glyceraldehyde-3-phosphate (G3P), NAD⁺, inorganic phosphate (Pᵢ) |
| **Product** | 1,3-bisphosphoglycerate (1,3-BPG), NADH |
| **Enzyme** | Glyceraldehyde-3-phosphate dehydrogenase |
| **Phase** | Energy payoff (시작) |
| **NADH** | +1 (× 2 = 2 total per glucose) |

## 특징

- **산화 + 인산화**: G3P를 산화하여 나온 에너지로 고에너지 acyl-phosphate 결합 형성
- **NADH 생성**: 세포질의 NAD⁺가 환원됨 → 이 NADH가 ETC에 전달될 때 shuttle 필요
- 1,3-BPG는 **고에너지 acyl-phosphate 결합** 포함 → Step 7에서 ATP 합성에 사용

## NAD⁺ 재생의 중요성

- 이 반응은 NAD⁺를 소모하는데, NADH에서 NAD⁺를 재생하지 못하면 glycolysis 중단
- 산소 있을 때: 미토콘드리아 ETC → NADH 산화
- 산소 없을 때: 발효(fermentation)로 NAD⁺ 재생
