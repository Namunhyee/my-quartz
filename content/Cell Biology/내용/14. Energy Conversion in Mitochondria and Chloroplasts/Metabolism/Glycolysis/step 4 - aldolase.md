# Step 4 - Aldolase

## 반응

**Fructose-1,6-bisphosphate → DHAP + Glyceraldehyde-3-phosphate**

```
Fructose-1,6-bisphosphate  ──(Aldolase)──→  DHAP + G3P
        (C6)                                  (C3) + (C3)
```

## 핵심 정보

| 항목 | 내용 |
|------|------|
| **Reactant** | Fructose-1,6-bisphosphate |
| **Product** | Dihydroxyacetone phosphate (DHAP), Glyceraldehyde-3-phosphate (G3P) |
| **Enzyme** | Aldolase |
| **Phase** | Energy investment (분할 단계) |
| **ATP** | 0 |

## 특징

- C₆ 분자를 두 개의 C₃ 분자로 **분할**
- DHAP는 Step 5에서 G3P로 전환되어 glycolysis 진행
- 이 단계부터 탄소 수가 반으로 → 이후 계산은 ×2
