# Mitochondria — Map of Contents

## 개요

**미토콘드리아(Mitochondria)**는 진핵세포에서 ATP를 대량 생산하는 핵심 에너지 전환 소기관이다. 세포질 부피의 최대 20%를 차지하며, 단순히 ATP 공장에 그치지 않고 다양한 대사·신호 기능을 수행한다.

### 막 구조

| 구획 | 특징 |
|------|------|
| **Outer mitochondrial membrane** | Porin(β-barrel)을 통해 이온 및 5,000 dalton 이하 소분자에 자유 투과성. Intermembrane space의 pH/이온 조성이 세포질과 유사 |
| **Intermembrane space** | Outer/inner membrane 사이 (~20–30 nm). 세포질과 유사한 환경 |
| **Inner mitochondrial membrane** | 이온·소분자에 대한 diffusion barrier. ATP/ADP, pyruvate 등 선택적 transport protein에 의해서만 통과 가능 |
| **Crista membrane** | Inner membrane의 깊숙이 접힌 부분. Electron-transport chain과 ATP synthase 위치. 단백질 밀도 매우 높음 (단백질 75%, 지질 25%) |
| **Matrix** | 미토콘드리아 내부. Citric acid cycle 효소, mtDNA, ribosome, 다수의 대사 효소 포함. 단백질 농도 >500 mg/mL |

### Cardiolipin

![[Cell Biology/출처/md converted/Chapter 14/_page_9_Figure_7.jpeg]]
*Figure 14-11: Cardiolipin 구조. 4개의 fatty acid chain을 가진 이중 phospholipid.*

- **Cardiolipin**: 미토콘드리아 막에 특이적인 2-headed phospholipid (fatty acid chain 4개)
- 산화적 인산화 단백질 및 ATP 수송과 밀접하게 상호작용
- Cristae의 고곡률 막 구조를 지지

## 미토콘드리아의 다양한 기능

| 기능 | 설명 |
|------|------|
| ATP 생산 | 산화적 인산화 (주요 기능) |
| NAD⁺ 재생 | Glycolysis 등에 필요한 NAD⁺ 공급 |
| 생합성 전구체 공급 | Citric acid cycle 중간산물 → 아미노산, 뉴클레오타이드, 지방산 합성 |
| Heme 및 Iron-sulfur cluster 합성 | 호흡과 핵 게놈 안정성에 필수 |
| 세포 신호 | Ca²⁺ 완충, ROS 생성·신호 |
| Apoptosis 조절 | Cytochrome c 방출 → 세포자살 cascade |
| 지질 생합성 | Cardiolipin, phosphatidylethanolamine 등 합성 |

## 역동적 특성

![[Cell Biology/출처/md converted/Chapter 14/_page_5_Figure_2.jpeg]]
*Figure 14-7: 효모 세포에서 미토콘드리아 reticulum의 동적 변화 (fission, fusion).*

미토콘드리아는 **정적인 bean 모양**이 아니라 지속적으로 이동·분열(fission)·융합(fusion)하는 동적 구조이다.

- **Fission**: dynamin-related GTPase가 helical oligomer를 형성 → GTP 가수분해 → constriction → 분열
- **Fusion**: outer membrane fusion (GTPase) + inner membrane fusion (dynamin-related protein), 각각 별도 기계 필요
- 기능 불량 미토콘드리아: fission 지속 + fusion 상실 → 과잉 단편화 → [[Cell Biology/내용/13. Intracellular Membrane Traffic/endocytosis/Autophagy|**mitophagy**]] (Pink1-Parkin pathway)로 제거

## 단백질 수입

미토콘드리아 단백질의 대부분(~99%)은 핵 유전자로 암호화되어 세포질 리보솜에서 합성된 후, [[Cell Biology/내용/12. Intracellular Organization and Protein Sorting/Protein import into Mitochondria/000_Protein import into Mitochondria_MOC|TOM/TIM 복합체를 통해 post-translational 방식으로 수입된다]]. 수입 에너지원으로는 ATP(cytosolic hsp70), 막전위([[Cell Biology/내용/14. Energy Conversion in Mitochondria and Chloroplasts/Metabolism/Electron Transport Chain/Chemiosmotic coupling|Chemiosmotic coupling]]으로 유지되는 H⁺ gradient), 산화환원 전위(Mia40)가 사용된다.
