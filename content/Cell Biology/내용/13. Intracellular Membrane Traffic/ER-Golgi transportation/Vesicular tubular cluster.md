# Vesicular tubular cluster

## 개요

**Vesicular tubular cluster**는 [[Cell Biology/내용/12. Intracellular Organization and Protein Sorting|ER]]-derived transport vesicle이 서로 fusion하여 형성하는 intermediate compartment이다. 이들은 ER과 [[Golgi apparatus_MOC]] 사이의 transport container로 기능한다.

## 형성 과정

![[_page_18_Figure_9.jpeg]]

### 1. [[COPII]] Vesicle Budding

**ER exit site에서**:
- Protein cargo가 COPII-coated vesicle에 package
- Multiple vesicle이 동시에 형성
- Rapid budding

### 2. Coat Shedding

Vesicle이 ER에서 분리된 직후:
- COPII coat rapidly disassemble
- Naked vesicle 생성
- Fusion-competent state

### 3. [[homotypic fusion vs heterotypic fusion|Homotypic Fusion]]

![[Cell Biology/출처/md converted/Chapter 13/_page_18_Picture_2.jpeg]]

**[[homotypic fusion vs heterotypic fusion|Homotypic fusion]]**:
- ER-derived vesicle끼리 fusion
- Matching [[SNARE]] set 사용:
  - 양쪽 membrane에 v-SNARE와 t-SNARE 모두 존재
  - Symmetrical interaction
- Multiple vesicle의 successive fusion

### 4. Cluster Formation

Fusion의 결과:
- Convoluted structure 형성
- Interconnected tubular and vesicular elements
- Electron microscope에서 distinctive appearance
