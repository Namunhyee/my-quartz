# 8번

## 발문

SNARE를 통한 membrane 결합과정에 대한 설명으로 맞는 것은? (3점)

## 선지

① Lipid bilayer 중 세포질에 노출된 쪽이 새로운 bilayer를 형성한다.
② 결합에 사용된 SNARE 복합체(V-&T-SNARE)는 사용 후 분해된다.
③ Hemifusion 단계에서 새로 형성되는 bilayer는 소포 쪽에서만 유래된다.
④ Stalk 형성 단계에서 lipid bilayer 중 세포질에 노출된 면이 stalk을 형성한다.
⑤ 소포와 target 세포막 사이의 물분자가 완전히 제거된 후 SNARE가 pairing 할 수 있다.

# 해설

**정답: ①**

[[Cell Biology/내용/13. Intracellular Membrane Traffic/Vesicle transportation/Vesicle Fusion/SNARE|SNARE-mediated membrane fusion]] 단계 분석:

**Fusion 단계 복습:**
1. Tethering → Docking (v-SNARE + t-SNARE trans-complex)
2. **Stalk 형성**: 두 막의 **외엽(outer leaflet, 세포질 노출 쪽)**이 먼저 접근·융합 → lipid stalk
3. **Hemifusion**: 외엽끼리 합쳐진 상태 → hemifusion diaphragm
4. **Pore 형성**: 내엽(lumenal leaflet)도 융합 → 내용물 방출

- **①: Stalk 단계에서 세포질에 노출된 외엽(outer leaflet)이 먼저 새로운 bilayer 중간 구조를 형성** ✓
- ②: 사용된 trans-SNARE complex는 분해되지 않고 **NSF/α-SNAP**에 의해 **해리(disassembly)** 후 재사용 ✗
- ③: Hemifusion 단계의 새 bilayer는 **소포와 target membrane 양쪽** 외엽이 합쳐진 것 ✗
- ④: ①과 같은 내용이지만 표현이 달라 혼동 유발 — ①이 더 정확 ✗
- ⑤: 물분자의 완전한 제거는 SNARE pairing의 전제 조건이 아님 ✗
