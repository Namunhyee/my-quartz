# 27번

## 발문

미토콘드리아 단백질 수입(import) 과정에서 빈칸을 채우시오.

세포질에서 미토콘드리아로 수입되는 단백질은 N-말단에 양전하를 띠는 __(A)__를 가지고 있다. 외막을 통과할 때는 __(B)__ 채널을 이용하고, 내막을 통과할 때는 __(C)__ 채널을 이용한다. 내막을 통과하려면 __(D)__에 의해 생성된 __(E)__가 필요하다.

# 해설

**빈칸 정답:**

- **(A): MTS (Matrix Targeting Sequence)** — 양으로 하전된 N-말단 신호 서열 (amphipathic α-helix)
- **(B): TOM (Translocase of Outer Membrane)** — Tom20, Tom22가 MTS 인식; Tom40이 채널
- **(C): TIM23 (Translocase of Inner Membrane 23)** — Matrix 행 단백질 수입 경로
- **(D): ATP synthase / 전자전달계 (respiratory chain)** — 미토콘드리아 내막 전위차 생성
- **(E): 막전위 (membrane potential, Δψ)** — 내막의 전기화학적 구배 (~-180 mV, 내부 음성)

**전체 경로 요약:**
세포질 (MTS 인식) → **TOM** 통과 → IMS → **TIM23** 통과 (Δψ 구동) → Matrix 진입 → mtHsp70(ATP 소비)으로 완전 수입 → MPP가 MTS 절단
