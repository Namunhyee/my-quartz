# 17번

## 발문

제시된 hydropathy plot에 대한 설명으로 적절한 것은? (3점)
![[_page_3_Figure_18.jpeg]]


## 선지

- ① (B) 단백질은 7개의 알파 헬릭스를 갖을 수 있다.
- ② Plot의 양의 값은 물분자와의 친화도가 높은 것을 의미한다.
- ③ (A) 단백질에서 3개의 알파 헬릭스(α-helix)가 세포막을 통과할 수 있다.
- ④ 베타 쉬트의 바깥쪽은 lipid와 접촉하기 때문에 hydropathy로 판단하기 쉽다.
- ⑤ 양의 값이 여러 개 존재하는 것은 베타 쉬트(β-sheet)를 이루고 있음을 의미한다.

# 해설

**정답: ①**

[[Cell Biology/내용/10. Membrane Structure/Hydropathy plot|Hydropathy plot]] 분석:

- **①: (B) plot에 양의 피크가 7개 → 7개의 transmembrane α-helix 예측 가능 (예: GPCR 계열 7-pass 막단백질)** ✓
- ②: Hydropathy plot의 **양(+)의 값 = 소수성(hydrophobic)** 높음 → 물과 친화도 낮음. 물 친화도 높음은 음(-)의 값 ✗
- ③: (A) plot에 양의 피크 1개-> 소수성 부분 1구역 -> 1개의 transmembrane α-helix.
- ④: β-barrel의 **막-접촉 소수성 잔기**는 hydropathy로 예측하기 **어려움** (개별 β-strand의 소수성 잔기가 alternating 패턴으로 배열되어 있어 연속적 양의 피크가 나타나지 않음) ✗
- ⑤: 여러 개의 양의 피크 = **여러 α-helix** 막 통과 (β-sheet가 아님) ✗
