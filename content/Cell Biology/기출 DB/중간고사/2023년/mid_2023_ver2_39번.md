# 해설

**미토콘드리아 Matrix 단백질 수입 과정 (에너지원 포함):**

**[준비 단계] 세포질 — 에너지: 불필요(수동 인식)**
- 합성된 단백질의 N-말단 **MTS(matrix targeting sequence)**를 세포질 chaperone(**Hsp70, Hsp90**)이 인식
- Chaperone이 단백질의 비정상적 응집 및 접힘을 방지하며 미토콘드리아 표면으로 안내
- 에너지: **ATP** (chaperone 활성 유지에 필요, 그러나 이 단계는 준비 단계)

**[1단계] 외막 통과 — TOM complex — 에너지: 없음(수동)**
- Tom20·Tom22가 MTS를 인식
- **Tom40** 채널을 통해 외막 통과 → IMS 공간으로 진입
- 에너지: **불필요** (농도 구배 또는 전기화학적 구배에 의한 수동 이동)

**[2단계] 내막 통과 — TIM23 complex — 에너지: 막전위(Δψ)**
- IMS의 Tim50이 MTS를 인식 → Tim23 채널로 안내
- 내막을 통과하는 데 **막전위(Δψ, ~-180 mV, 내부 음성)**가 양전하 MTS를 전기적으로 matrix 방향으로 당김
- 에너지: **Δψ (membrane potential)**

**[3단계] 완전 수입 — mtHsp70 — 에너지: ATP**
- Matrix의 **mtHsp70 (Tim44에 고정)**이 수입 중인 단백질 사슬을 반복적으로 잡아당김(ratchet mechanism 또는 power stroke)
- 각 주기마다 **ATP 가수분해** 에너지 소모
- 단백질이 완전히 matrix 내로 당겨짐

**[마무리 단계]**
- **MPP (Matrix Processing Protease)**: MTS를 절단하여 성숙 단백질 생성
- **mtHsp60/Hsp10**: 단백질의 올바른 최종 접힘 보조 (ATP 소모)

**에너지원 요약:**
| 단계 | 에너지원 |
|------|---------|
| 세포질 chaperone | ATP |
| TOM 통과 | 불필요 |
| TIM23 통과 | **막전위 (Δψ)** |
| mtHsp70 완전 수입 | **ATP** |
| mtHsp60 접힘 | ATP |
