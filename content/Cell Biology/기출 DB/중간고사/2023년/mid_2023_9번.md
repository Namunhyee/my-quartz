# 해설

**정답: ①**

[[Cell Biology/내용/13. Intracellular Membrane Traffic/endocytosis/Autophagy|Autophagy]] 분석:

이미지는 이중막 구조의 **autophagosome**을 보여줌.

- **①: Autophagosome이 lysosome과 융합하여 autolysosome이 되어야 분해가 시작됨 → 이 융합 과정에 SNARE 단백질(예: STX17, VAMP7) 필요** ✓
- ②: 내부 물질들은 분해되지만, **autophagosome의 외막**은 lysosome 막과 융합하고 **내막과 내용물**이 분해됨. "세포막을 제외"라는 표현은 부정확 ✗
- ③: 외부 물질 분해는 **endocytosis(이종소화)**의 역할. Autophagy는 **자가소화** — 세포 내부 물질(손상된 소기관, 단백질 응집체 등) 분해 ✗
- ④: Autophagosome 형성 후 **lysosome과 융합해야만** 내용물 분해 시작 → 즉시 분해 불가 ✗
- ⑤: **Hurler's disease**는 lysosomal hydrolase(α-L-iduronidase) 결핍 → **lysosome 내 기질 축적** 문제. Autophagosome **형성** 자체는 정상적으로 가능 ✗
