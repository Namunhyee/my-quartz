# 해설

소포가 목적지 막에 도달하는 과정에서 **Rab 단백질**의 역할:

**[[Cell Biology/내용/13. Intracellular Membrane Traffic/Vesicle transportation/Transportation Mechanisms/Rab Cycle|Rab Cycle]]:**

1. 세포질에서 **Rab-GDP**는 **GDI(GDP Dissociation Inhibitor)**와 결합한 상태로 존재
2. 특정 막(donor membrane)의 **GEF**가 GDP → GTP 교환 → **Rab-GTP** 활성화
3. Rab-GTP가 소포 막에 삽입 → **Rab effector** 단백질 recruit → 소포 이동(motor protein 등)
4. Rab-GTP가 target membrane에도 존재 → **테더링(tethering)** 유도
5. Target membrane의 **GAP**가 GTP 가수분해 → Rab-GDP → GDI 결합 → 막에서 해제 → 재활용

**핵심:**
- 각 소기관마다 **고유한 Rab 단백질**이 있어 "주소지" 역할
- GDI는 Rab을 비활성화가 아닌 **세포질로 수송**하는 역할
- 활성화된 Rab-GTP는 vesicle과 target membrane **모두**에 존재 가능
