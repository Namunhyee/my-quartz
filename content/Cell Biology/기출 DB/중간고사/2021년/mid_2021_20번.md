# 해설

**세포질 → 미토콘드리아 단백질 수송 3가지 경로 (목적지에 따라):**

**① Matrix 수송 (TOM + TIM23 경로)**
- 양으로 하전된 **N-말단 matrix targeting sequence(MTS)** 보유
- **TOM(Translocase of Outer Membrane)**: 외막 통과
- **TIM23(Translocase of Inner Membrane 23)**: 내막 통과 → matrix 진입
- 에너지: **막전위(Δψ)** + **ATP**(mtHsp70 구동)
- Matrix 도착 후 MTS는 **matrix processing protease(MPP)**가 절단

**② Inner membrane 삽입 (TOM + TIM22 경로)**
- 내막 캐리어 단백질(multi-pass: ADP/ATP carrier 등)
- TOM → IMS chaperone(Tim9·Tim10) → **TIM22** → 내막 측방 삽입
- 에너지: **막전위(Δψ)만** 필요 (ATP 불필요)

**③ Intermembrane space (IMS) 진입 (MIA 경로)**
- Cysteine 잔기 보유, MTS 없음
- TOM small channel 통과 → **Mia40**에 의해 disulfide bond 형성 → IMS에 포획
- 에너지: **산화적 접힘(oxidative folding)** 에너지
