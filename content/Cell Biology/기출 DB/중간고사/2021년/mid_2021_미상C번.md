# 해설

**v-SNARE:**
- Vesicle membrane에 위치한 SNARE 단백질 (예: synaptobrevin/VAMP)
- Target membrane의 t-SNARE와 **trans-SNARE complex** 형성 → 막 융합 유도
- "v" = vesicle, 소포 막에 존재

**Rab effector:**
- 활성화된 **Rab-GTP**가 recruit하는 downstream 효과기 단백질
- 종류: tethering factor(소포를 target membrane에 고정), motor protein adaptor(소포 이동), kinase 등
- 각 Rab 단백질마다 고유한 effector set 존재 → 소포의 목적지 특이성 부여

> 복기자 주: "영영풀이를 보고 무엇을 의미하는지 맞추는 문제 → v-SNARE, Rab effector"
