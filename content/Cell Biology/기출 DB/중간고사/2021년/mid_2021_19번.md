# 19번

## 발문

세포질의 단백질을 inner mitochondrial membrane과 intermembrane space로 단백질 수송하는 방법을 서술하시오.

# 해설

**미토콘드리아 각 구획으로의 단백질 수송:**

**1. Inner membrane 단백질 수송:**

경로 A (TIM23 → lateral release):
- TOM → TIM23 채널 → **내막 내 측방 방출(lateral release)**
- 내막의 소수성 stop-transfer 서열이 채널에서 내막 지질층으로 lateral exit

경로 B (TIM22 pathway):
- 내막 캐리어 단백질(예: ADP/ATP 캐리어)에 특화
- TOM → IMS chaperone → **TIM22** 복합체 → 내막 삽입
- 막전위(Δψ) 필요, matrix 이입 없음

**2. Intermembrane space(IMS) 단백질 수송:**

경로 (MIA pathway):
- **Signal sequence 없음**, IMS targeting을 위한 cysteine-rich 서열 보유
- TOM을 통해 IMS로 진입 → **Mia40 (oxidoreductase)**가 disulfide bond 형성 도움 → 단백질 접힘 → IMS에 트랩

또는 TIM23 경로로 먼저 matrix에 넣은 후 **OXA translocase**로 다시 내막 삽입 후 처리.
