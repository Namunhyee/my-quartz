# 해설

**방향성 기전: RanGDP와 RanGTP의 공간적 분포 차이**

[[Cell Biology/내용/12. Intracellular Organization and Protein Sorting/Nuclear Pore Complex/Transport through NPC/Ran-GTP cycle|Ran-GTP Cycle]]:

| | 핵(Nucleus) | 세포질(Cytoplasm) |
|--|------------|-----------------|
| **Ran 상태** | Ran-GTP (high) | Ran-GDP (high) |
| **조절 인자** | RanGEF (Chromatin-bound) | RanGAP |
| **역할** | cargo 해리 | cargo 결합 |

**핵 수입(import) 메커니즘:**
1. 세포질에서 Importin이 NLS 신호를 가진 cargo와 결합 (Ran-GDP 상태)
2. NPC를 통해 핵 내로 이동
3. 핵 내 **Ran-GTP**가 Importin에 결합 → cargo **해리**
4. Importin-RanGTP 복합체가 세포질로 이동 → RanGAP가 GTP 가수분해 → RanGDP → Importin 해리 → 재사용

**핵 수출(export) 메커니즘:**
1. 핵 내에서 Exportin이 **Ran-GTP**와 cargo(NES 신호 포함)와 3중 복합체 형성
2. NPC를 통해 세포질로 이동
3. 세포질 **RanGAP**에 의해 Ran-GTP → Ran-GDP → 복합체 해리 → cargo 방출

→ **RanGEF는 핵에, RanGAP는 세포질에 존재**하여 Ran-GTP/GDP 농도 구배를 만들고, 이 구배가 수송의 방향성을 결정.
