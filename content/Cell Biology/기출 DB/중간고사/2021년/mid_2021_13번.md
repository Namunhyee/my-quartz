# 해설

**코트 단백질 종류와 이동 경로:**

| 코트 단백질 | 이동 경로 | 방향 |
|------------|---------|------|
| **COPII** | ER → Golgi (순행, anterograde) | ER 출구 → VTC → CGN |
| **COPI** | Golgi → ER (역행, retrograde) / Golgi 내 역행 | 회수 경로 |
| **Clathrin (AP1)** | TGN → Lysosome (endosome 경유) | |
| **Clathrin (AP2)** | Plasma membrane → Early endosome | 내포작용 |

- **ER → Golgi**: **COPII** 코트 단백질 (Sar1-GTP에 의해 조립 시작, Sec23/24 inner coat, Sec13/31 outer coat)
- **Golgi → Plasma membrane (분비 경로, TGN 이후)**: Clathrin-independent 경로이며, TGN에서의 sorting은 복잡함. 구성적 분비(constitutive exocytosis)는 코트 단백질 없이 진행될 수 있음 / 또는 **Clathrin/AP1** (TGN → endosome/lysosome 방향)
