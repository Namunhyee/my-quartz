# 1번

## 발문

아래는 해당작용(glycolysis) 과정에 발생되는 중간체들(intermediates)을 나열하였다. 발생 되는 순서대로 나열한 것을 고르시오. (6점)

- I. Fructose 6-phosphate
- II. 3-Phophoglycerate
- III. 2-Phophoglycerate
- IV. Dihydroxyacetone phosphate
- V. Glyceraldehyde 3-phosphate

# 해설

**정답(순서): I → IV → V → II → III**

Glucose → G6P → F6P(I) → F1,6-BP → DHAP(IV) + G3P(V) [DHAP→G3P via triose phosphate isomerase] → 1,3-BPG → 3-PGA(II) → 2-PGA(III) → PEP → Pyruvate
