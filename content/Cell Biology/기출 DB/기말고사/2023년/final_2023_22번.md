# 해설

**1. Tubulin-Sequestering protein: Stathmin (Op18)**

- **Stathmin (Oncoprotein 18, Op18)**은 자유 α-β tubulin 이합체 2개와 결합하여 **세포질의 가용 tubulin pool 감소** → 미세소관 중합 억제
- 인산화 시 tubulin 방출 → 중합 촉진
- 세포 분열 조절, 암세포에서 과발현

**2. Microtubule catastrophe promoting protein: MCAK (Kinesin-13 / KIF2C)**

- **MCAK (Mitotic Centromere-Associated Kinesin)**은 kinesin superfamily 단백질이나 이동하지 않고 미세소관 end에 결합
- ATP 가수분해를 통해 미세소관 end의 tubulin을 **강제 해중합** → catastrophe 유발
- 유사분열 시 잘못 부착된 염색체 교정에 관여
- (또는 **XKCM1**, **kin I kinesin** 등으로도 알려짐)

**3. Arp2/3와 microtubule에서 비슷한 역할을 하는 protein: γ-TuRC (γ-tubulin Ring Complex)**

- **Arp2/3 복합체 (actin)**: 기존 actin 필라멘트의 측면에서 새 필라멘트를 분지(branching) 핵화
- **γ-TuRC (microtubule)**: 중심체(centrosome)에서 새 미세소관의 **(-) end를 핵화(nucleation)** — Arp2/3처럼 새 필라멘트/소관의 핵형성을 담당
- 공통점: 둘 다 **새로운 필라멘트(소관) 형성을 핵화**하는 역할

> [!note] γ-TuRC는 microtubule nucleation에서, Arp2/3는 actin branched nucleation에서 같은 역할군(핵화 단백질)으로 분류됨.
