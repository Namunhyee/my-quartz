# 5번

## 발문

제시글은 citric acid cycle의 intermediate(중간체)들의 일부이다. Citric acid cycle 과정에서 **먼저 발생되는 순서대로** 나열한 것을 고르시오. (E, 2점)

## 선지

*(A) malate*
*(B) succinate*
*(C) α-ketoglutarate*

- ① A, B, C
- ② A, C, B
- ③ B, A, C
- ④ B, C, A
- ⑤ **C, B, A**

# 해설

**정답: ⑤ C, B, A**

TCA cycle 순서 (citrate 합성부터):

1. Acetyl-CoA + OAA → **Citrate** (Citrate synthase)
2. → Isocitrate (Aconitase)
3. → **α-ketoglutarate (C)** + CO₂ (Isocitrate dehydrogenase, NADH)
4. → Succinyl-CoA + CO₂ (α-ketoglutarate dehydrogenase, NADH)
5. → **Succinate (B)** + ATP (Succinyl-CoA synthetase)
6. → Fumarate (Succinate dehydrogenase, FADH₂)
7. → **Malate (A)** (Fumarase)
8. → OAA (Malate dehydrogenase, NADH) → 다음 회전

**순서: C → B → A = ⑤**
