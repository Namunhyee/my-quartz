## 발문

그림에서 **왼쪽 그림과 오른쪽 그림의 차이점을 설명하시오.** (서술형, 5점)

*(그림: 왼쪽 = branched actin network (Arp2/3 기반, lamellipodia), 오른쪽 = unbranched linear actin filaments (formin 기반, filopodia/stress fiber))*

# 해설

**왼쪽 그림 — Branched actin network:**

- **Arp2/3 복합체**에 의해 형성
- 기존 필라멘트의 측면(70° 각도)에서 새 필라멘트가 분지(branching)
- **NPF(N-WASP, WAVE)**에 의해 Arp2/3 활성화 → **Rac1/Cdc42 GTPase** 신호
- 결과 구조: **dendritic/branched network**
- 세포 구조: **Lamellipodia** (넓고 납작한 돌출 구조)
- 기능: 세포 이동의 protrusion 단계

**오른쪽 그림 — Unbranched linear filament:**

- **Formin(mDia)** 에 의해 형성
- 새 G-actin을 (+) end에 지속적으로 추가하며 **직선으로 연장(barbed end protection)**
- **RhoA GTPase** 신호로 formin 활성화
- 결과 구조: **unbranched, parallel filament bundles**
- 세포 구조: **Filopodia** (가는 돌기), **Stress fiber** (장력 유지), **Cytokinetic ring**
- 기능: 방향 탐색, 세포 이동 견인, 세포 분열

**핵심 차이 요약:**

| | 왼쪽 (Branched) | 오른쪽 (Linear) |
|--|---------------|----------------|
| 핵화 단백질 | **Arp2/3 + NPF** | **Formin** |
| GTPase | Rac1, Cdc42 | RhoA |
| 구조 | 분지형 망상 | 직선 다발 |
| 세포 구조 | Lamellipodia | Filopodia, Stress fiber |
| (+) end 상태 | 분지점에서 새 필라멘트 생성 | Formin이 (+) end 보호하며 연장 |
