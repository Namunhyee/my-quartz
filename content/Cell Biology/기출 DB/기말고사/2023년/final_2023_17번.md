# 해설

**b = Cytoplasmic dynein (세포질 다이닌)**

---

**구조:**
- 2개의 **heavy chain** (AAA+ ATPase 도메인, motor domain 포함) + intermediate/light chain
- **ATP 가수분해**로 구동되는 two-headed motor
- **Dynactin 복합체**(p150glued, Arp1 등)와 함께 작동하여 cargo와 결합

---

**이동 방향:**
- 미세소관의 **(−) end 방향 (retrograde)** — 세포 말단 → 세포체(중심체, 핵) 방향

---

**주요 기능:**

1. **세포 내 소포·소기관 수송 (retrograde transport)**
   - 내포소체(endosome), 리소좀 → 세포체 방향 수송
   - 신경 세포: 시냅스 말단 → 세포체 방향의 역행 축삭 수송

2. **핵 위치 결정**
   - 중심체와 함께 핵을 세포 중심으로 유지

3. **방추사 형성 (mitosis)**
   - 유사분열 시 염색체·방추사 극 조직화에 관여
   - 방추사 극을 분리하는 힘 생성 (역방향 힘)

4. **섬모/편모 운동 (ciliary dynein)**
   - Axonemal dynein: 9+2 axoneme 내 이중 미세소관 사이를 슬라이딩 → 섬모 굽힘

5. **소포 이동 (주요 기능 요약)**
   - Kinesin: (+) end → 세포 말단 (anterograde)
   - **Dynein: (−) end → 세포 중심 (retrograde)** ← b 단백질
