# 해설

**a–f 각 구조 이름 및 특징:**

| 기호 | 이름 | 특징 |
|------|------|------|
| **a** | **Lamellipodia (판상위족)** | 세포 leading edge의 넓고 납작한 돌출. Branched actin (Arp2/3). Rac1 의존. 세포 이동 protrusion |
| **b** | **Filopodia (사상위족)** | 가늘고 긴 침 형태 돌출. Unbranched parallel actin (formin). Cdc42 의존. 방향 탐색 |
| **c** | **Stress fiber (스트레스 섬유)** | 굵은 actin-myosin II 다발. 세포 내 장력 유지. Focal adhesion에 연결. RhoA 의존 |
| **d** | **Contractile ring (수축환)** | 세포 분열 시 세포질 분열(cytokinesis). 적도면에 actin-myosin II 환 형성 → 수축 → 세포 분리 |
| **e** | **Microvilli (미세융모)** | 세포 표면 흡수 면적 증가. 소장 상피 brush border. Parallel actin bundle (villin, ezrin 결합) |
| **f** | **Sarcomere (근절)** | 골격근 수축 기본 단위. Actin thin filament + Myosin II thick filament 교대 배열. Z-disk 사이의 단위 |

> 그림의 정확한 배열에 따라 a–f 매칭이 달라질 수 있음. 위 표는 일반적인 6종류 actin array 기준.
