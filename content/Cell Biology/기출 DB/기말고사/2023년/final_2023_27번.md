# 27번

## 발문

Fibroblast가 합성하고, connective tissue를 구성하며, 포유류에서 가장 많이 발현하는 단백질로 whole-body protein content의 25% 정도를 차지하고 있으며, type 1 alpha 1 chain이 돌연변이를 일으키는 경우 Osteogenesis imperfecta가 유발되는데 관여하는 **extracellular matrix는 무엇인가?** (E, 1점)

# 해설

**정답: Collagen (콜라겐)**

---

**Collagen 특징:**

- **가장 풍부한 포유류 단백질**: 전체 단백질의 약 **25~30%** 차지
- **합성 세포**: 주로 **fibroblast**, 연골세포, 골세포
- **구조**: 3중 나선(triple helix) — Gly-X-Y 반복 서열 (X=Pro, Y=Hyp 빈번)
- **결합 조직(connective tissue)** 주요 구성 성분: 뼈, 연골, 힘줄, 피부, 기저막

**Type I Collagen:**
- 가장 풍부한 collagen 종류
- **COL1A1** (α1 chain) + **COL1A2** (α2 chain) → [α1]₂[α2]₁ 삼합체
- 뼈, 피부, 힘줄에 풍부

**COL1A1 돌연변이 → Osteogenesis Imperfecta (OI, 골형성 부전증):**
- Type I collagen 구조 이상 → **뼈가 쉽게 부러지는 "유리뼈 질환"**
- Gly 대체 돌연변이가 가장 흔함 → triple helix 형성 불가

**Collagen 합성 경로 요약:**
세포질 리보솜 → ER (signal peptide, 3-수산화 Pro, Hyp) → Golgi (propeptide 추가) → 세포 외 분비 → procollagen peptidase 절단 → tropocollagen → fibril 형성(cross-linking)
