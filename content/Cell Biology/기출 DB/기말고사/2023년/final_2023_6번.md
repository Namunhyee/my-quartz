# 6번

## 발문

제시글은 citric acid cycle에 관여하는 enzyme(효소)들의 일부이다. Citric acid cycle 과정에서 **먼저 작용하는 순서대로** 나열한 것을 고르시오. (E, 2점)

## 선지

*(A) Fumarase*
*(B) Succinate dehydrogenase*
*(C) Aconitase*

- ① A, B, C
- ② A, C, B
- ③ B, A, C
- ④ B, C, A
- ⑤ **C, B, A**

# 해설

**정답: ⑤ C, B, A**

TCA cycle 효소 작용 순서:

| 단계 | 효소 | 반응 |
|------|------|------|
| 2 | **Aconitase (C)** | Citrate → Isocitrate |
| 3 | Isocitrate dehydrogenase | Isocitrate → α-KG (NADH) |
| 4 | α-KG dehydrogenase | α-KG → Succinyl-CoA (NADH) |
| 5 | Succinyl-CoA synthetase | Succinyl-CoA → Succinate (ATP) |
| 6 | **Succinate dehydrogenase (B)** | Succinate → Fumarate (FADH₂) |
| 7 | **Fumarase (A)** | Fumarate → Malate |
| 8 | Malate dehydrogenase | Malate → OAA (NADH) |

**순서: C(2단계) → B(6단계) → A(7단계) = ⑤**
