# 3번

## 발문

제시글은 glycolysis(해당작용) 반응에 관여하는 enzyme(효소)들의 일부이다. Glycolysis 과정에서 **먼저 작용하는 순서대로** 나열한 것을 고르시오. (E, 2점)

## 선지

*(B) Aldolase*
*(C) Enolase*

② A, C, B
③ **B, A, C**

⑤ C, A, B

# 해설

**정답: ③ B, A, C**

Glycolysis 효소 작용 순서:

| 단계 | 효소 | 반응 |
|------|------|------|
| 1 | Hexokinase | Glucose → G-6-P |
| 2 | Phosphoglucose isomerase | G-6-P → F-6-P |
| 3 | Phosphofructokinase | F-6-P → F-1,6-BP |
| **4** | **Aldolase (B)** | F-1,6-BP → DHAP + G-3-P |
| 5 | Triose phosphate isomerase | DHAP ↔ G-3-P |
| 6 | GAPDH | G-3-P → 1,3-BPG |
| **7** | **Phosphoglycerate kinase (A)** | 1,3-BPG → 3-PG (ATP 생성) |
| 8 | Phosphoglycerate mutase | 3-PG → 2-PG |
| **9** | **Enolase (C)** | 2-PG → PEP |
| 10 | Pyruvate kinase | PEP → Pyruvate (ATP 생성) |

**순서: B(4단계) → A(7단계) → C(9단계) = ③**
