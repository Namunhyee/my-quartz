# 15번

## 발문

Filament capping과 filament dynamics에서 그것의 효과에 관한 그림이다. **그림에서 초록색 라인과 붉은색 라인의 차이를 설명하시오.** (서술형, 5점)

*(그림: 시간-filament 길이 그래프. 초록색 = capping 없음(정상), 붉은색 = (+) end capping 처리)*

# 해설

**초록색 라인 (capping 없음, control):**

- G-actin 단량체가 (+) end와 (-) end **양쪽으로 자유롭게 중합·해중합**
- Steady state에서 treadmilling 발생 → 필라멘트 총 길이 유지
- 전형적인 S자형 중합 곡선: lag phase → exponential growth → plateau

**붉은색 라인 (+ end capping 처리, 예: CapZ/gelsolin):**

- **(+) barbed end가 capping 단백질에 의해 차단** → (+) end에서의 G-actin 추가 불가
- **(-) pointed end만** 통해 해중합 진행 → 필라멘트가 짧아지거나 성장 정체
- **전체 F-actin 양 감소** (actin 단량체 pool이 증가)
- Treadmilling 불가 → steady state 수준이 낮아짐

**핵심 차이 요약:**

| | 초록색 (uncapped) | 붉은색 (+ end capped) |
|--|-----------------|---------------------|
| (+) end 성장 | 가능 | **차단** |
| Treadmilling | 발생 | 불가 |
| 최종 filament 길이 | 정상 plateau | **더 낮은 plateau 또는 감소** |
| G-actin pool | Steady state | 증가 |

**생물학적 의의:**
- 세포 내 capping protein(CapZ, Eps8 등)은 (+) end를 막아 actin 중합을 조절
- Gelsolin(Ca²⁺ 활성화): 필라멘트를 자르고 (+) end를 capping → 빠른 depolymerization 유도
